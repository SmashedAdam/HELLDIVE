Es gibt 2 Versionen von diese Programme.
In ./src/main.py können Sie mit CLI-Modus das Programm (Version 3) laufen.
In ./src/modules/alt.py können Sie mit GUI-Modus das Programm laufen.

In ./docs stehen die Dokumentation und das originale Nassi-Shneiderman Diagramm.
Sie können auch die HTML-Datei auf https://charyan.github.io/nsdmaker/ öffnen.
Sie können das vollständige Repository auf dieser GitHub-Seite anzeigen: https://github.com/SmashedAdam/HELLDIVE
